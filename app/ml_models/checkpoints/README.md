# Model Checkpoints

This directory contains trained model checkpoints.

## Structure
- `checkpoint_epoch_10.pt` - Training checkpoint at epoch 10
- `checkpoint_epoch_20.pt` - Training checkpoint at epoch 20
- `best_model.pt` - Best performing model
- `latest_model.pt` - Most recent model

## Model Versions
- v1.0.0 - Initial release (2024-01-15)
- v1.1.0 - Improved accuracy (2024-02-01)
- v2.0.0 - Multi-class support (2024-03-01)
